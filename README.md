## Clase 15 - 10

### Trabajo con instancias EC2

Se crea una instancia EC2 desde AWS, se recomienda usar micro pues la capa gratuita dura 1 hora y se puede borrar.


Se comienza con:
 - `sudo apt update`
 - `sudo apt upgrade -y`


Para instalar MySQL:

`sudo apt install mysql-server`
